Features

Everything Icarus can do

A full breakdown of every feature — from simple key presses to pixel-reactive automation.

Actions

9 action types

Every step your macro can take, chained in any order with full drag-and-drop reordering.

⌨️

Key & Mouse Input

Three input actions cover every scenario: Key Press (tap and release), Key Down (hold), and Key Up (release). Works with keyboard keys, mouse buttons, and controller buttons. Combine down/up pairs to hold a key for exactly as long as you need.

key_press key_down key_up mouse_click
  • Full keyboard support including function keys and numpad
  • Mouse left, right, and middle click
  • XInput and DualShock controller buttons
  • Reassign any input after creation with Set Input
🖱️

Mouse Movement

Two dedicated mouse movement actions give you precise cursor control. Move Mouse (Position) moves to an absolute screen coordinate — instant or animated with humanization. Move Mouse (Location) simulates a relative directional look movement, ideal for camera control in games.

mouse_move mouse_look
  • Multi-monitor support with per-display coordinates
  • Humanized curves with configurable path randomness
  • Speed variance and overshoot for natural movement
  • Instant mode for zero-delay repositioning
  • 8-direction compass or freeform angle for mouse-look
💬

Type Phrase

Type any string of text as simulated keystrokes. Choose from three speed presets (Fastest, Balanced, Safest) or set a custom per-character delay. Instant mode pastes the entire phrase in one OS call for scripts that need to bypass keystroke detection.

  • Supports all printable ASCII characters
  • Three speed presets: Fastest (1ms), Balanced (3ms), Safest (6ms)
  • Custom speed in milliseconds per character
  • Instant paste mode for speed-critical scenarios
⏱️

Delay & Loop

Insert precise waits between actions. Use a fixed delay or enable random mode to pick a duration between a min and max value — useful for bypassing anti-macro detection. The Loop action replays the entire macro sequence continuously.

  • Fixed delays with ms precision
  • Random delay range (min/max) for human-like timing
  • Loop step repeats the entire macro indefinitely
  • Runtime duration display updates live in the editor

Conditions

5 trigger conditions

Control exactly when a macro fires. Stack multiple conditions and add OR groups for complex trigger logic.

🔄

Always On

The macro runs continuously as long as it is enabled. No trigger needed — ideal for persistent background tasks.

⬇️

While Key Held

Fires repeatedly while a key is held down. Supports Repeat mode so the macro loops for as long as the key stays pressed.

🔀

Toggle With Key

One key press turns the macro on; another turns it off. The macro remembers state between presses.

▶️

Action From Key

Executes once per key press. Optional safe mode prevents re-triggering while the macro is still running.

🎨

Pixel Color Match

Watches a screen pixel or nearby radius area. Triggers the moment the color matches your target within a set tolerance. Supports all connected displays.

OR / Alt Condition Groups

Add alternate condition groups so one macro can be triggered by completely independent scenarios. The macro fires if the primary condition group or any alt group is satisfied — no need to duplicate macros.

Profiles & Organization

Stay organized at scale

📁

Profile System

Group macros into profiles for different games, applications, or contexts. Switch the active profile with a global hotkey or from the HUD. Each profile has its own enable/disable hotkey so you can arm or disarm an entire set instantly.

  • Unlimited profiles, each saved as its own JSON file
  • Per-profile enable/disable hotkeys
  • Import and export individual macros between profiles
  • Rename, duplicate, and delete profiles from the UI
🏷️

Macro Categories

Assign each macro to a category — Combat, Utility, Movement, Crafting, System, or Uncategorized. Filter the macro list by category to find what you need quickly even in large profiles.

  • 6 built-in categories
  • Per-macro toggle bind for instant enable/disable
  • Per-macro window focus lock
  • Run count display in the HUD

HUD Overlay

Always-on-top information overlay

A floating panel that stays on screen during use — fully configurable, click-through, and multi-monitor aware.

📍

Positioning

Snap to any corner of any connected display. Set a custom width, height, and opacity. Border toggle included.

🖱️

Click-Through Mode

Enable click-through so the HUD never interferes with your game or application. Interactions pass straight through.

🔧

Configurable Panels

Toggle individual panels: title bar, profile name, clock, uptime, macro summary, active macro list, pixel-watch status, and controller state.

📊

Run Counts

Optionally show how many times each macro has fired this session next to its name in the HUD list.

More

Extra capabilities

🎮

Controller Support

Bind macros to XInput (Xbox-compatible) and DualShock controllers. The runtime monitors controller state alongside keyboard and mouse.

🪟

Window Focus Lock

Per-macro window restriction. Assign a window title substring — the macro only fires while that window is in the foreground.

🔭

Pixel Watcher (Reactive)

Reactive macros live on their own page. A configurable poll interval (1–20 ms) keeps CPU usage low while staying responsive.

💾

Import / Export

Share macros as portable JSON files. Import a macro from any file on disk directly into your active profile.

Release History

Changelog

Every release, what changed, and what was fixed.

v1.0.0 July 2025 Latest
New Features
Drag-and-drop action reordering with floating ghost card and drop indicator line
Pixel Color Match condition (replaces "On Pixel Change") with improved label clarity
Reactions page poll interval changed from free-entry to dropdown (1, 2, 3, 4, 5, 10, 20 ms)
Action labels renamed: Move Mouse (Position), Move Mouse (Location)
OR / Alt condition groups — one macro, multiple independent triggers
Per-macro toggle bind hotkey
Per-macro window focus lock
Macro categories: Combat, Utility, Movement, Crafting, System, Uncategorized
HUD fully configurable — opacity, border, click-through, per-panel toggles, custom size
Multi-monitor support across all pixel watch and mouse move features
Import / Export macros as portable JSON files
XInput and DualShock controller input backend
Improvements
Widget cache system — macro editor reuses widgets instead of rebuilding on every refresh
Runtime duration display updates live in the action editor
Random delay range (min/max) for human-like timing
Humanized mouse movement with path randomness, speed variance, and overshoot
Type Phrase instant mode for speed-critical scenarios
v0.9.0 Placeholder – 2025 Previous
New Features
Initial Reactive Macros page with pixel watcher
Profile management with enable/disable hotkeys
HUD overlay with basic panel support
Mouse movement actions with animation support
Type Phrase action with speed presets
Bug Fixes
Fixed drag-and-drop direction being limited to ±1 step at a time
Fixed widget state going stale after editing a step without rebuilding
Fixed scroll container coordinate offset in drop calculations
v0.5.0 Placeholder – 2025 Previous
New Features
Core macro engine with keyboard and mouse input
Conditions: Always On, While Key Held, Toggle With Key, Action From Key
Actions: Key Press, Key Down, Key Up, Mouse Click, Delay, Loop
Basic macro list and editor UI
JSON-based storage for macros, profiles, and settings

What it does

Everything you need, nothing you don't

Build complex automation with a drag-and-drop editor, conditional triggers, reactive pixel watching, and multi-profile support.

⌨️

Keyboard, Mouse & Controller

Bind macros to any key, mouse button, or controller input across three backends including XInput and DualShock.

🎯

Pixel Color Match ⭐ Premium

Watch a pixel or area on screen for a specific color. Trigger a macro the instant it matches.

⚙️

Powerful Action Editor

Chain key presses, mouse moves, delays, typed phrases, loops, and more. Reorder with drag-and-drop.

📁

Profiles

Create separate profiles for different games or tasks. Switch instantly with a hotkey.

🖥️

HUD Overlay ⭐ Premium

A floating, click-through overlay showing active macros, profile, uptime, and more.

🖱️

Advanced Mouse ⭐ Premium

Humanized curves, path randomness, speed variance, and overshoot simulation for natural movement.

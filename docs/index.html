What it does

Everything you need, nothing you don't

Build complex automation sequences with a drag-and-drop editor, conditional triggers, reactive pixel watching, and multi-profile support.

⌨️

Keyboard, Mouse & Controller

Bind macros to any key, mouse button, or controller input across three input backends including XInput and DualShock.

🎯

Pixel Color Match

Watch a pixel or area on screen for a specific color. Trigger a macro the instant it matches — perfect for game-state reactions.

⚙️

Powerful Action Editor

Chain key presses, mouse moves, delays, typed phrases, mouse-look, loops, and more. Reorder with drag-and-drop.

📁

Profiles

Create separate profiles for different games or tasks. Switch instantly with a hotkey or from the HUD overlay.

🖥️

Customizable HUD

A floating, click-through overlay shows your active macros, profile, uptime, and more. Fully configurable — size, corner, opacity.

🪟

Window Focus Lock

Restrict individual macros to only run when a specific window is in focus. Per-macro, per-profile — no global toggles needed.

ICARUS MACRO MAKER – RELEASES
==============================

Place your release ZIP here and name it:
  IcarusMacroMaker_v1.0.zip

To create it:
  1. Zip the entire IcarusMacroMakerv4_cleaned folder
  2. Name the ZIP file IcarusMacroMaker_v1.0.zip
  3. Move it into this releases/ folder
  4. Commit and push to your GitHub repo

The download button on index.html already points to:
  docs/releases/IcarusMacroMaker_v1.0.zip

For larger releases, consider using GitHub Releases instead
and update the href in index.html to point to the release asset URL.

# Icarus Macro Maker

Icarus Macro Maker (IMM) is an open-source macro creation tool for Windows.
It allows users to build keyboard/mouse automation sequences with a clean UI,
profile system, and runtime engine.

## Features
- Macro editor with conditions + actions
- Profile system
- Runtime engine with key/mouse listeners
- CustomTkinter UI
- Fully open-source

## Installation
Releases > Unzip > Launch EXE
